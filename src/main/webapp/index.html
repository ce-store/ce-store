Welcome to the IBM® Controlled Natural Language Processing Environment (ICNLPE).

ICNLPE is also referred to as 'CE Store' or the 'CE environment' in this documentation.

  • You can exercise the various REST APIs of the CE Store, e.g. against /ce-store or whatever you have named your web application.
  • There is some limited documentation for the CE Store.
  • You can access the user interface to the CE Store via the Engineering Panel.

CE Store is a web-application that can be installed into a Java JEE web-container, such as those provided by IBM® WebSphere® Application Server, IBM® WebSphere® Application Server Liberty Profile, IBM® WebSphere® Application Server Liberty Core and Apache Tomcat.

Note :

  • IBM WebSphere Application Server is also referred to as 'WebSphere Application Server' in this documentation.
  • IBM WebSphere Application Server Liberty Profile and IBM WebSphere Application Server Liberty Core are also referred to as 'WebSphere Liberty Application Server' in this documentation.
  • Apache Tomcat is also referred to as 'Tomcat' in this documentation.
