How to set up CE Store from war files

Pre-requisite: install Tomcat (any recent version) and ensure that it is started
  1. Drop war files into your existing {tomcat}/webapps folder:
    • CeStoreWeb.war
    • CeEngineeringPanel.war
    • CeResourcesWeb.war
  2. Navigate in a browser (Chrome or Firefox) to: http://localhost:8080/CeEngineeringPanel (for Tomcat)
  3. In the Engineering Panel (in your browser) in the actions pane (on the left) click on the correct model to load it for your exploration
Note:
  • A similar approach can be taken with WebSphere Application Server and WebSphere Liberty Application Server.
  • The default ports for these servers may be different.
  • For example, for WebSphere Liberty Application Server, place the wars in wlp/users/servers/{your-server}/dropins and use port 9080.
