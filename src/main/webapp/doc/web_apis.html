application/json text/plain
/sources GET List all sources
  • style=summary
    (default)
A JSON list of sources in their summary form. A plain text list of all CE sentences for every source plus the meta-model CE sentences for each of these sources
  • style=full
A JSON list of sources in their full form.
/sources/{id} GET Get details for the source identified by {id}
  • style=full
    (default)
The JSON details for full form for the source. A plain text list of all CE sentences for the source plus the meta-model CE sentences for each of these sources
  • style=summary
The JSON details for summary form for the source.
DELETE Delete the source identified by {id}
This is a cascade delete. All sentences within the source will be deleted, plus any entities created from those sentences, including:
  • Models
  • Concepts
  • Properties
  • Instances
  • Rules
  • Queries
The deletion of the entity occurs if the deletion of the sentence(s) results in no remaining sentence referencing the entity in question. 		tbc A plain text description indicating that the source has been deleted

  • style=summary
    (default)
A JSON list of sentences in their summary form. A plain text list of all CE sentences for the source
  • style=full
A JSON list of sentences in their full form.

Sentences

(description of sentences is needed here)

URL Method Description Parameters HTTP request 'Accept' header
application/json text/plain
/sentences GET List all sentences.
Optionally specify one or more type and validity parameters which will act as filters
  • Sentence type:
    • Model
    • Fact (all)
    • Fact (normal)
    • Fact (qualified)
    • Pattern (rule or query)
    • Rule
    • Query
    • Annotation
    • Command
  • Sentence validity:
    • Valid
    • Invalid
If multiple type and/or validity parameters are supplied, sentences are listed after the combination of filters have been applied. This is an example of combining filters to retrieve valid model sentences.
  • style=summary
    (default)
  • type=model | fact | fact-normal | fact-qualified | pattern | rule | query | annotation | command
  • validity=valid | invalid
A JSON list of sentences in their summary form. A plain text list of all CE sentences
  • style=full
  • type=model | fact | fact-normal | fact-qualified | pattern | rule | query | annotation | command
  • validity=valid | invalid
A JSON list of sentences in their full form.
POST Save the specified CE sentences in the CE Store, and by parsing them generate all derived entities
  • ceText={text}
    The CE sentences to be processed
  • action={val}
    The action to be taken when processing the sentence
    default {val}=save
  • setName={val}
    The optional name for this set of sentences (used when creating the source)
    default {val}=null
tbc A plain text summary of the CE sentences that were processed and saved
